📄 Sobre este entorno

Esta página es servida a través de un proxy inverso Nginx que distribuye las peticiones entre dos backends en modo round-robin. Ambos backends comparten el mismo contenido web mediante un volumen Docker compartido. La red interna es completamente aislada: los backends no son accesibles directamente desde el exterior, solo a través del proxy.
